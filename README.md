## Installation

```
git clone https://github.com/kojix2/umap
cd umap
rake compile
rake install
```

* OpenMP is required for multithreading.

## Usage

```ruby
Umap.umap(pixels, nthreads: 4)
```

Parameters and default values

```markdown
- method               : :annoy (another option is vptree)
```
